# Documentation

This directory contains all documentation for the AI Adoption MASEM dissertation.

## Directory Structure

| Folder | Contents |
|--------|----------|
| `01_literature_search/` | Search strategy, database coverage |
| `02_study_selection/` | Inclusion/exclusion criteria, screening protocol |
| `03_data_extraction/` | Coding manual, construct harmonization, AI pipeline docs |
| `04_methodology/` | MASEM methods, model specification, Bayesian & network protocols |
| `05_manuscript/` | Writing timeline and drafting notes |
| `06_decisions/` | Decision log for methodological choices |

## Key Documents

- **Coding Manual** (`03_data_extraction/coding_manual.md`): Comprehensive guide for extracting correlation matrices and study metadata
- **Construct Harmonization** (`03_data_extraction/construct_harmonization.md`): TAM↔UTAUT↔AI construct mapping rules
- **Model Specification** (`04_methodology/model_specification.md`): Three competing structural models
- **Bayesian Protocol** (`04_methodology/bayesian_masem_protocol.md`): Sabherwal priors and MCMC settings
- **Network Protocol** (`04_methodology/network_analysis_protocol.md`): MAGNA methodology
